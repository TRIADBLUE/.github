# TRIADBLUE

We build cloud-based business platforms for small and local businesses.

Our ecosystem is built around eight platforms, each focused on a distinct part of running and growing a business. They integrate where it makes sense and stay independent where it doesn't.

## Platforms

| Platform | What it does |
|---|---|
| **businessblueprint.io** | Business growth platform for SMBs — apps for marketing, communications, listings, reputation, advertising, and AI coaching. |
| **hostsblue.com** | Domains, hosting, email, and website builder. |
| **swipesblue.com** | Payment processing — the single payment surface for the entire ecosystem. |
| **scansblue.com** | Website auditing and scanning. |
| **builderblue².com** | AI-powered vibe coding platform — standalone and integrated with hostsblue. |
| **LINKBlue.systems** | Monitors website health across the ecosystem. |
| **linksblue.network** | Agent-facing infrastructure (GitHub proxy, MCP server, AI conversation archive). |
| **Console.Blue** | Internal operations interface. |

## businessblueprint apps

Two suites and three standalone tools:

- **/ compass suite** — communications: / promote, / engage, / respond, / post
- **/ anchor suite** — local presence: / elevate, / optimize, / publish, / amplify
- **/ connect** — CRM (the data hub)
- **/ convert** — lead capture and conversion
- **/ Coach Blue** — AI business coach
- **/ assess** — Digital IQ assessment (free)
- **/ scan** — website scanner

## Engineering principles

- Production-ready over placeholder. No "coming soon" pages.
- Single sources of truth for shared data (app registries, brand colors, pricing).
- All payment processing through swipesblue. No third-party processors elsewhere in the ecosystem.
- Brand consistency is non-negotiable.
